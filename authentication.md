# Аутентификация

Все методы требуют обязательные query-параметры:

| Параметр | Тип | Описание |
|----------|-----|----------|
| `account` | string | Логин аккаунта Яндекс.Директ |
| `client` | string | Логин клиента |

## Пример

```
GET /webHooks/getCampaignsList?account=example_account&client=example_client&from=2026-01-01&to=2026-01-20
```

## Форматы ответов

Все методы возвращают JSON в едином формате:

**Успешный ответ:**
```json
{
    "status": "ok",
    "data": {}
}
```

**Ошибка:**
```json
{
    "status": "error",
    "error": "Описание ошибки"
}
```

## Обязательные параметры

- **account** - логин аккаунта Яндекс.Директ
- **client** - логин клиента

Эти параметры требуются для всех методов API, если не указано иное в описании конкретного метода.
